# 我是怎么用的 —— 设计哲学

*这是本仓库的「非技术」说明，讲的是**为什么**这套环境是现在这个样子：我怎么用 Claude、
怎么用键盘驱动整台电脑，以及 `fr0m` 和那些 hook 到底是干什么的。*

🌏 English → [`how-i-work.md`](how-i-work.md) · 🗺️ 快捷键速查表 → [`../cheatsheet.html`](../cheatsheet.html)

这不只是一堆配置文件，而是一套**工作方式**。它由两个核心理念支撑：
**先立规矩、再信任 AI**，以及**用键盘而不是鼠标驱动电脑**。

---

## fr0m 是一切的核心

我把 Claude 当作一个**聪明但健忘**的合作者。聪明，所以我放心把真活交给它；健忘，所以我
绝不让「意图」只活在会滚走的聊天记录里。**每个项目都从 `/fr0m` 开始**——它会立下四份文档，
这四份文档本身**就是项目的记忆与法律**：

| 文档 | 是什么 | 扮演的角色 |
|------|--------|-----------|
| **Principal.md** | 宪法——最终目标 + 硬性限制。 | **归我所有。** Claude 不能私自改写；要动它，必须是一次明确的决定。 |
| **Plan.md** | 唯一真相源——永远保持对当前工作的最新描述。 | 动手前先重读；每个新决定都立刻并入这里。 |
| **AOL.md** | 仅追加日志——给所有做过的事打上时间戳的轨迹。 | 持久的记忆。只追加、从不修改，历史无法被悄悄改写。 |
| **Errors.md** | 每一个错误以及它是怎么解决的。 | 同一堵墙不撞第二次。 |

一句话哲学：**把意图外化成持久的、仅追加的文档，工作就变得可治理、可审计、可续接**——
对我、对 Claude、对任何 clone 下来的人都一样。聊天只是对话；fr0m 把它变成一个有宪法、
有完整记录的项目。这也是为什么**本仓库自己**就是用 fr0m 治理的——看它的 `Principal/Plan/Errors` 就知道。

## hook 让纪律自动生效

没人执行的规矩只是愿望。所以纪律被写进了**每一轮都会运行**的 hook 里——我不必去记，
Claude 也无法跑偏：

- **`fr0m-guard`** 会悄悄拒绝那些会破坏体系的操作：改写仅追加日志、背着我编辑宪法、
  或者在 commit 里把 Claude 署名为共同作者。遇到看不懂的情况它**放行**（fail-open），
  所以它在守规矩的同时不碍事。
- **`fr0m-rules`** 在**项目处于治理状态时**，于每一轮开头重新注入这些常驻规则，
  让上面这套世界观始终摆在 Claude 面前——绝不会「聊三句就忘」。
- **`ip-guard`** 是一道个人安全护栏（本仓库默认关闭）。重点不在于具体检查什么，
  而在于：**护栏只需设一次，之后就一直默默保护你**，无需你再操心。

## `/check` —— 「给我看」，而不是「跟我说」

我是视觉型的人，很少想看大段解释文字。**`/check` 会把 Claude 产出的东西渲染成一个
自包含的 HTML 页面，并立刻在 Wave 里弹出来。** 产出由此变成一个**可看的成品**——
一张表、一张图、一个仪表盘、本仓库自己的 `cheatsheet.html`——而不是要我在脑子里想象的段落。
（`blocks` 输出风格是它的小尺度版本：每条回复都以一条分隔线结尾，让每一轮读起来都是干净、
可扫读的「块」。）

## Claude 在我一天中的位置

`wave .` 一条命令就摆好一个「编码驾驶舱」：左边是我的 shell，**中间是并排的 Claude 和 Codex**，
右边是文件树。我让多个 agent 并行干活，并把设置调成适合硬核工作的状态——`effort: high`、
`advisor: opus`（在难题上给第二意见）、再配一套看着舒服的自定义主题。Claude 是一个在我
工作区里有工位的队友，而不是一个我要去访问的网站。

## 键盘就是命令面

桌面的摆法让我**永远不用找窗口**——**一个键，一个地方，全是肌肉记忆：**

- **Caps Lock → Wave**，在任何地方都行。键盘上最没用的键，现在一按就跳到 agent 所在的地方。
  这是我整套设置的「Home 键」。
- **Option + 数字**在桌面之间切换；**Option + 字母**一键召唤一整个应用**工作区**——
  `⌥W` 微信、`⌥N` Notion、`⌥L` 飞书、`⌥F` Firefox，每个都在自己专属、带标签的 space 上，
  已经替我打开并排好。`⌥M` / `⌥R` 一次性最小化、再恢复整个 space。
- **yabai 自动平铺每一个窗口**（不用拖、不重叠），**skhd 按方向移动焦点**——
  `⌥`+方向键移动注意力，`⇧⌥`+方向键重新排列。鼠标是后备，不是默认。

贯穿始终的一条线：**规矩设一次，接好线让它自己守住，然后把注意力花在活儿上**——
而不是花在记「东西在哪」「上周决定了什么」。
